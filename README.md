
# Belajar expressJS

First, run the development server:


## Run Locally

Clone the project

```bash
  git clone https://github.com/dedikurniawanx/expressJS.git
```

Go to the project directory

```bash
  cd expressJS
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm start
  or
  pnpm dev
```
